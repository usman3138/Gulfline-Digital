# Gulfline Digital

Landing page for Gulfline Digital — websites built for tutors, trainers, and freelancers across the UAE.

**Live site:** [gulflinedigital.com](https://gulflinedigital.com)

## About

Gulfline Digital offers fast, affordable, custom-coded websites (no templates) for solo professionals — tutors, trainers, coaches, and freelancers — who need a credible online presence without agency overhead.

- Custom-designed, up to 5 pages
- Mobile-first, fast-loading
- WhatsApp click-to-chat built in
- Basic on-page SEO
- Delivered in 7 business days
- Bilingual: English / Arabic (with RTL support)

## Tech Stack

Single-file static site — no build step, no dependencies.

- HTML5 / CSS3 / vanilla JavaScript
- Fonts: Space Grotesk, Inter, Noto Kufi Arabic (Google Fonts)
- No frameworks, no bundler — deploys as-is

## Project Structure

```
index.html   → entire site (markup, styles, and scripts in one file)
```

## Deployment

Hosted on Hostinger. Deployment is connected via Git — pushing to `main` auto-deploys to production.

```bash
git add .
git commit -m "your change description"
git push origin main
```

## Local Development

No build tools required. Just open `index.html` directly in a browser to preview changes.
